# apollo-graphql-pokeapi

An GraphQL implementation of the PokeAPI using Apollo-GraphQL.

### Sample Query

```
{
  getPokemon(id: 249) {
    id
    name
    abilities {
      is_hidden
      ability {
        name
      }
    }
    stats {
      base_stat
      stat {
        name
      }
    }
  }
  getDescription(id: 249) {
    flavor_text
    language {
      name
    }
    version {
      name
    }
  }
}
```
